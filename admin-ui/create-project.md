# Create a project

The first step in being able to interact with Atlas CMS is to create a project. In Atlas CMS, a project represents an independent environment that serves as a container for all data related to it.

Once logged in, you will be able to see the list of all the projects you have access to, whether created by you or shared by others.

To create a new project you must select the Add Project card, and enter information such as the Display Name, the Api Name you want to use in API calls and an optional Description.

::: tip
The Api Name is unique within Atlas. If you use generic names it is possible that another project previously created by other users is already using it, and Atlas CMS will ask you to enter a different name.
:::

The created project will be added to the list of your projects and you can select it to enter the dashboard and start configuring it and managing the contents.

::: tip
Every project you create is assigned the free tier. You can change plans at any time and switch to a paid plan by accessing the Settings → Billings section.
:::
