# Temp

## 

```
GET /temp/{project}/context
```

### Path Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `project` | string | Yes | - |

### cURL Example

```bash
curl -X GET "https://my-project.atlascms.io/temp/{project}/context" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json"
```
