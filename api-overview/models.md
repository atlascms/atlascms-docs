# Models

Models define the structure of your content in Atlas CMS. Each model represents a content type with its associated fields, and determines how data is stored and exposed via the API.

Models are created through the [Visual Model Builder](/admin-ui/modeling-data) in the Admin UI. Once defined, they become available as API endpoints for CRUD operations on content.

For detailed information on creating and configuring models, see [Modeling Data](/admin-ui/modeling-data).

## API Endpoints

Model and component metadata can be accessed via the Content Types API. See the [API Reference](/api-reference/#content-types) for the complete list of endpoints.
