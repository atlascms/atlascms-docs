# Getting Started

Welcome to Atlas CMS! This documentation guides you through all technical and non-technical topics related to the setup, use, and customization of your project. It covers both the administration interface aspects and the full API documentation.

## Want to jump right in?

Atlas CMS offers three main ways to interact with its features:

1. **Admin UI:** A web-based interface for managing content, models, and assets visually. Ideal for content creators and marketers.

2. **APIs:** REST and GraphQL APIs for programmatic access to all Atlas CMS functionalities.

3. **SDKs:** Software Development Kits for JavaScript and .NET Core to simplify integrations.
