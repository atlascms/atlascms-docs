# Use Cases

Atlas CMS is designed to support a wide range of use cases for modern web and application development.

## Websites and Blogs

Power your websites and blogs with structured content. Define models for articles, authors, categories, and media. Use the Admin UI for content creation and the APIs for headless delivery to any frontend framework.

## E-commerce

Manage product catalogs, categories, and content with flexible modeling. Connect your e-commerce frontend to Atlas CMS via REST or GraphQL APIs for dynamic product information, descriptions, and media.

## Mobile Applications

Deliver content to iOS and Android apps through our APIs. Use localization features to serve content in multiple languages based on user preferences.

## Headless Architecture

Atlas CMS excels in headless setups. Store and manage content in Atlas, then deliver it to:

- Static site generators (Next.js, Nuxt, Astro, etc.)
- Mobile apps (React Native, Flutter)
- IoT devices
- Third-party platforms

## Multi-channel Content

Create content once and publish everywhere. Use the same content models to feed websites, mobile apps, digital signage, and more.

## Configurations and Lookups

Use single-collection models for configuration data, form dropdown options, or any scenario requiring a single document rather than a list.
